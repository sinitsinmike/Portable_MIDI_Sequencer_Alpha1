# Alpha1 Workflows (RU)
Портативный MIDI-секвенсор — Public Alpha 1

---

## 1. Обзор
Этот документ описывает, **как Alpha1 используется на практике**, с фокусом на:
- рабочие процессы записи
- поведение в реальном времени
- логику меню и редактирования
- ограничения в реальном времени

Документ дополняет User Manual и Quick Reference.

---

## 2. Линейный режим записи

[STOP]
  |
  | установка BPM / Track / Mode = Linear
  v
[Нажата REC]
  |
  | 4 такта count-in
  v
[ЗАПИСЬ]
  |
  | MIDI-события записываются один раз вдоль таймлайна
  | Метроним активен
  |
[STOP]
  |
  v
[PLAY]

---

## 3. Loop-режим записи

[STOP]
  |
  | Mode = Loop
  | установка длины лупа
  v
[Нажата REC]
  |
  | 4 такта count-in
  v
[LOOP ЗАПИСЬ]
  |
  | overdub включён по умолчанию
  | автоматический возврат в начало лупа
  |
[STOP]

---

## 4. Меню и редактирование

Главный экран
  |
  | Нажатие энкодера
  v
Режим редактирования (*)
  |
  | Вращение энкодера
  v
Изменение значения
  |
  | Нажатие энкодера
  v
Применить / Выход

---

## 5. Ограничения во время записи
Во время записи заблокированы:
- Save / Load / Clean
- Изменение длины лупа
- Переключение режима секвенсора

---

## 6. Переключение треков
FF → Track +1  
REW → Track -1

Мгновенное переключение без подтверждения.
