# Известные ограничения (Alpha3)

-   Максимальный размер проекта ограничен одним flash-сектором (4096
    байт)
-   Максимальное количество MIDI-событий ≈ 500 на проект
-   Хранилище использует фиксированный формат проекта (VERSION = 2)
-   Динамическое многосекторное хранение пока не реализовано

------------------------------------------------------------------------

# Архитектурное ограничение

Текущая версия хранения проектов использует фиксированный flash-сектор
на проект.

Это обеспечивает:

-   Простоту структуры
-   Детерминированность записи
-   Предсказуемость flash-операций

Однако это ограничивает максимальное количество событий в проекте.

В будущих версиях планируется внедрение динамического распределения
секторов.
